# @0disoft/openfeature-local-provider

Status: pre-implementation alpha skeleton

This package provides a local OpenFeature provider for JSON flag snapshots. The first
implementation supports schema version 1 parsing, static typed evaluation, missing flag
defaults, type mismatch error results, and a minimal OpenFeature provider adapter.

Deferred features include environment overrides, deterministic bucketing, audit JSON
Lines, replay fixtures, YAML, file watching, CLI, browser support, HTTP API, and database
integration.
